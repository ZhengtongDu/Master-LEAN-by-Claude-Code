# 第三章：类型论基础

## 本章概要

本章介绍 LEAN 背后的理论基础——依赖类型论。我们将从熟悉的集合论视角出发，逐步过渡到类型论的思维方式，理解"命题即类型"的深刻洞见。

---

## 3.1 从集合论到类型论：思维转变

### 内容点
- 集合论的基本概念回顾
- 类型论与集合论的区别
- 为什么类型论更适合计算机验证
- 类型论的历史：从 Russell 到 Martin-Löf

### Toy Case 简介
**类型 vs 集合**：对比集合论中 "x ∈ A" 与类型论中 "x : A" 的不同含义。

---

## 3.2 类型、项与判断

### 内容点
- 类型（Type）的概念
- 项（Term）与类型的关系
- 类型判断（Typing Judgment）
- Type Universe：`Type`、`Type 1`、`Type 2`...
- `Prop` 与 `Type` 的区别

### Toy Case 简介
**探索类型层级**：使用 `#check` 查看各种对象的类型，如 `#check Nat`、`#check Type`、`#check Type 1`。

---

## 3.3 函数类型与 Lambda 表达式

### 内容点
- 函数类型 `A → B`
- Lambda 表达式：`fun x => ...`
- 函数应用与求值
- 多参数函数与柯里化
- 隐式参数与显式参数

### Toy Case 简介
**构造函数**：定义一个简单的函数 `double : Nat → Nat`，并使用 `#eval` 测试。

---

## 3.4 依赖类型初探

### 内容点
- 什么是依赖类型
- 依赖函数类型（Π 类型）：`(x : A) → B x`
- 依赖对类型（Σ 类型）：`(x : A) × B x`
- 依赖类型的强大表达能力
- 长度索引向量示例

### Toy Case 简介
**向量类型**：定义长度索引的向量类型 `Vec α n`，体会依赖类型如何在类型级别编码信息。

---

## 3.5 Curry-Howard 同构：命题即类型，证明即程序

### 内容点
- Curry-Howard 同构的核心思想
- 命题对应类型
- 证明对应项（程序）
- 逻辑连接词的类型论解释
- 为什么这个对应如此重要

### Toy Case 简介
**蕴含即函数**：证明 `P → Q` 等价于构造一个从 `P` 到 `Q` 的函数。
